# DislocationDensityFileReader

!syntax description /UserObjects/DislocationDensityFileReader

`DislocationDensityFileReader` is used to provide dislocation densities on a per-grain basis to initialize grain dislocation densities.  Data is provided as a single column.  Input parameters include `file_name`, which is the name of the file to be read, and `lines_to_skip`, which will skip the specified number of header lines (for example, in case the user wishes to add metadata to the file in the header).
