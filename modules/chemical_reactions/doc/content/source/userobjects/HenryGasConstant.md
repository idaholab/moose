# HenryGasConstant

!syntax description /UserObjects/HenryGasConstant

## Description

Performs a calculation of the Henry gas coefficient for selected noble gas and salt systems.
The model is described in K. Lee, et al., ``Semi-empirical model for Henry's law constant of noble gases
in molten salt'', Scientific Reports (2024) 14:12847, https://doi.org/10.1038/s41598-024-60006-9.
