# HPC OnDemand

[HPC OnDemand](https://hpcondemand.inl.gov/pun/sys/dashboard), is a service provided by INL which
allows a user direct access to the resources contained within the HPC enclave via their web browser.

## Access

In order to utilize this service, you must first [request an account](https://inl.gov/ncrc).

Once your request has been accepted, and you have been given the necessary credentials provided by
the HPC team, you access the service via your web browser, with the link dependent on whether
you are on the INL network (INL employees):

- On INL network: go to [ondemand.hpc.inl.gov](https://ondemand.hpc.inl.gov) and log in with your INL HPC username and password.
- Not on INL network: go to [hpcondemand.inl.gov](https://hpcondemand.inl.gov/) and log in with your INL HPC username and PIN + token.

## Dashboard

The Dashboard is your homepage when using HPC OnDemand. The following represents some of the more useful features, and is not an exhaustive list:

| Description | Navigation |
| :- | :- |
| View/download/upload files using a GUI file explorer | Files -> Home Directory |
| List your active jobs | Clusters -> Active Jobs |
| Interactive shell access | Clusters -> *Machine* Shell |
| Launch Linux desktop with visualization | Apps -> Linux Desktop |
| Launch VSCode session | Apps -> VSCode |
