# NodeElemVariableResidualNorm

This Postprocessor derives from [VariableResidualNormBase.md] and computes the nonlinear residual norm $|r_i|$ of a variable $u$ on a single `NodeElem`, where $i$ is the single degree of freedom index for the variable $u$ on the `NodeElem`.
