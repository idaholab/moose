# MFEMHypreGMRES

## Summary

!syntax description /Solver/MFEMHypreGMRES

## Overview

Defines and builds an `mfem::HypreGMRES` solver to use as an iterative solver to solve the MFEM
equation system.

Not compatible with use on partially assembled equation systems.
