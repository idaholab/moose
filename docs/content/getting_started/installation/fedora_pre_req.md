## Prerequisites
Install the following using dnf

```bash
  sudo -E dnf install gcc \
gcc-c++ \
gcc-gfortran \
tcl \
tk \
findutils \
make \
freeglut-devel \
libXt-devel \
libX11-devel \
m4 \
blas-devel \
lapack-devel \
git
```

Download and install one our redistributable packages according to your version of Fedora.
